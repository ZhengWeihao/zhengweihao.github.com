Java常用操作笔记
---

* 查看字节码文件：
  * javap -c -v CLASS_FILE_NAME [ > TEXT_FILE]
    * 常见指令：
      * iconst_0：将int类型的常量0压入栈
      * istore_2：将int类型的值存入局部变量2
      * iload1：从局部变量1中装载变量值想
      * iadd：执行加法运算
